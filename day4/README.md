\# Day 4: Multi-Step AI Agent (Planning Agent)



\## Objective



Design a multi-step AI agent that: - Breaks complex queries into steps -

Executes them sequentially - Shows intermediate outputs - Logs execution



\## Concept



User Input → Planning → Execution → Output



\## Features



\-   Supports sum, average, max, min

\-   Handles multi-step queries

\-   Shows intermediate steps



\## Project Structure



day4/ │ ├── agent.py ├── tools.py ├── logger.py ├── main.py ├── logs.txt

└── README.md



\## Example



Input: Find average of 10,20,30 and summarize



Output: Extract numbers → Compute average → Summarize

python main.py
